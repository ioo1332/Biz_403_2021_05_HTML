Flex를 이용한 화면 구현

초창기 HTML은 위에서 아래방향으로 진행되는 Layout이 주를 이뤘다

가로방향으로 content를 배열하는 layout이 필요하게 된다

여러가지 Hack(tip)들이 등장을하고 다양한 방법이 동원된다

표준 HTML사용법과는 다소 거리가 있어서

여러가지 문제들을 일으키고

원하는대로 layout이 만들어지지 않았다

대표적인 방법으로 div box를 floating하여 배열하는 방법이 사용되었다

대표적인 box model

우리나라 만세

box model tag를 float로 설정을 하면

box width에 따라화면에서 한라인에 여러 tag를 배열할수 있다

float로 만든 layout은 floating 되기를 원하지않는 tag까지 영향을 미치게된다

inline-block

float를 사용하여 layout을 구성하던 HTML디자이너들에게 반가운 소식이 전해진다

HTML5의 표준에 box model을 변경할수있는 속성이 추가된다

display 속성

기본적인 box model을 변경하여 적용할수있다

block->inline

inline->block

div tag에 display를 inline으로 설정했더니

갑자기 block성질을 버리고 한라인에 여러개의 box를 배열할수있게 됐다

inline 속성값은 box의 width를 설정할수 없다

드디어 div box를 사용하여 원하는대로 width를 설정하고

한라인에 여러 박스를 배치할수 있게 되었다
